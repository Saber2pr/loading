# Loading

API:

```html
<html>
  <head>
    <script src="https://saber2pr.top/loading/index.min.js"></script>
  </head>
  <body>
    <script>
      // play loading.
      LOADING.init()
    </script>

    <script>
      // finish loading after 2 seconds.
      setTimeout(() => LOADING.destroy(), 2000)
    </script>
  </body>
</html>
```
